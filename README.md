# MediQueue
A comprehensive smart healthcare queue management with AI-powered symptom severity prediction using machine learning, real-time SMS notifications via Twilio, and optimized hospital resource allocation. Built with Node.js (Express), Python (Flask), and modern web technologies to streamline patient care and emergency response.

# MediQueue

## Getting Started

### Start the Server
To start the server, run:
```bash
npm start

```

### Start the Dectection system
To start the server, run:
```bash
python app.py
```
